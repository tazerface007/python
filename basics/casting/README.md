# Python Casting

Type casting is useful when you want to convert one datatypes in to another.

There are mainly two types of casting:

- Implicit casting
- Explicit casting


- Explicit casting
    - int(): Construct an integer number from an integer literal, a float literal, or a string literal.
    - float(): Construct a float number from an integer literal, a float literal or a string literal.
    - str(): Construct a string from a wide variety of data types, including strings, integer literal and float literals
